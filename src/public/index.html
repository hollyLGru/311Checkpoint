Routes:

/Login :

This is where students will log in. Will use the token

/register :

Students will go to this route to create an account. They will have to provide email and password

GET /myPANGEA :

This is where you can see ALL of the entries made to myPangea

/myPANGEA/:id :

You can get a single entry by the entry's ID. you can also edit the entry, delete an entry, and create a new entry. Each entry will have an ID, date, city, country, diary, and photo URL . POST for creating. PUT for editing old entries. DELETE to delete an entry

/myPANGEA/userentries/:UserId :

This returns ALL of the entries made by a specfic user by the userID. This will use GET

/myPANGEA/continent/:continent

This returns ALL of the entries made per continent which will be important because this is a travel diary. This will be GET
